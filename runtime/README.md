## Scope

This folder is for runtime-facing specification and ABI/runtime contracts.

Examples:

- runtime data structure layout expectations
- panic/runtime helper contracts
- string and array runtime semantics
- C ABI details that should not live in the language surface chapters

These documents are runtime-facing and should stay separate from the public language specification.
