# Overview

## Identity

Vek is a statically typed, whole-program compiled language with a prototype TypeScript frontend, a C backend, static dispatch, a tiny native support layer, and minimal native output.

## Core Goals

Vek is designed to optimize for:

- explicitness where it matters
- static dispatch
- minimal runtime complexity
- whole-program compilation
- tiny native binaries
- easy C interop
- straightforward semantics
- low ceremony syntax

## General Direction

Vek is heavily inspired by existing successful languages such as Rust/TypeScript and adopts useful ideas where they fit the language cleanly.
